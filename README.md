# Catálogo de Produtos Interativo

Este é um catálogo de produtos interativo desenvolvido com HTML, CSS, JavaScript e jQuery. Ele carrega dados de um arquivo JSON local via AJAX, permitindo visualização, busca, filtragem e ordenação dinâmica de produtos.

## Funcionalidades
- Carregamento de produtos via AJAX (botão "Carregar Produtos").
- Exibição em cards responsivos.
- Busca em tempo real por nome.
- Filtros por categoria.
- Ordenação por nome ou preço.
- Modal de detalhes para cada produto.
- Design responsivo para mobile, tablet e desktop.

## Estrutura do Projeto
- `index.html`: Estrutura da página principal.
- `css/style.css`: Estilos principais e responsividade.
- `js/main.js`: Lógica principal (carregamento AJAX, eventos).
- `js/produtos.js`: Funções para manipulação de produtos.
- `data/produtos.json`: Dados dos produtos.
- `assets/images/`: Imagens (placeholder).

## Como Usar
1. Abra o projeto no VS Code.
2. Instale a extensão "Live Server".
3. Clique com o botão direito em `index.html` e selecione "Open with Live Server".
4. Clique em "Carregar Produtos" para iniciar.

## Tecnologias
- HTML5
- CSS3
- JavaScript (jQuery)
- AJAX para carregamento de JSON local

## Notas
- Sem backend Java; usa apenas arquivos locais.
- Para adicionar mais produtos, edite `data/produtos.json`.
- Imagens são placeholders; substitua por URLs ou arquivos locais.
